# 金融行业面试真题库（中国 · 买方 / 卖方）

> 开源整理、**持续更新**。覆盖 投行 IBD / PE·VC / 二级研究 / **投资法务·合规**（本库主打）/ 风控 / 香港外资岗。
> 题目为**公开渠道整理 + 行业通识**，不含任何机构内部或保密信息。**欢迎 PR 补充真题**（走 [分享面经 Issue](../../issues/new/choose)）。
> 📱 公众号 ［待填］ 回复 `面经` 领**打包 PDF + 更新推送**。

## 一、通用 & 行为面（所有岗位必考）

- 为什么选金融 / 这个细分方向？为什么**买方 vs 卖方**？为什么这家机构？
- 用 STAR 讲一个你最有成就感的项目 / 最难的挑战。
- 职业规划？3–5 年后想成为什么样的人？
- 离职 / 换赛道原因？（诚实、正向、指向发展而非抱怨）
- 团队冲突 / 高压 / deadline 怎么应对？
- **反问环节**问什么加分（团队打法、这个岗位的成长路径、近期重点项目）。

## 二、投行 IBD

**技术**
- 三张报表如何联动？"折旧 +10、税率 40%"对三表的影响走一遍。
- 完整讲一遍 **DCF**：FCFF 怎么算、WACC 怎么算、终值两种方法、EV→股权价值的桥。
- **可比公司 vs 可比交易**：怎么选可比？常用倍数（EV/EBITDA、EV/Sales、P/E）及适用场景？
- **LBO** 核心？回报三来源（去杠杆 / EBITDA 增长 / 倍数扩张）、IRR 与 MOIC 区别？
- 为什么用 EV 而非股权价值？accretion/dilution（增厚/摊薄）怎么判断？
- 并购流程、协同效应、尽调关注点？

**案例 / 市场**
- 给一家公司估值你会怎么做？最近关注的一个 deal / IPO？对某行业的看法？

## 三、PE / VC（一级投资）

- 你怎么看一个项目？**投资逻辑框架**（赛道 / 团队 / 商业模式 / 壁垒 / 估值 / 退出）？
- 一级 vs 二级估值差异？早期项目怎么估？
- 业务 / 财务 / 法律**尽调**各自重点？发现红旗怎么办？
- 投资条款：**优先清算权 / 反稀释 / 对赌 / 回购 / 领售跟售** 怎么理解、风险在哪？
- **募投管退** 每一环关键？投后管理做什么？退出路径（IPO / 并购 / 老股转让）？
- 基金结构：GP/LP、管理费 2/20、**Carry** 怎么分？
- 讲一个你看过的案子，为什么投 / 不投？

## 四、二级研究（卖方 / 买方研究）

- 怎么分析一个行业（波特五力 / 产业链 / 供需格局）？
- 一个你看好的股票：逻辑、催化剂、风险、目标价怎么来的？
- 怎么建模、怎么估值、怎么跟踪基本面（关键指标）？
- 当前宏观 / 某板块观点？研究员如何产生 alpha？

## 五、投资法务 / 合规（本库主打方向）

**交易法律**
- PE/VC 投资协议核心条款（**SPA / SHA**）？对赌与回购的法律风险与可执行性？
- 交易结构设计考虑什么（税 / 监管 / 退出便利）？**股权代持 / VIE / 红筹** 法律要点？
- 法律尽调清单？发现瑕疵（诉讼 / 股权 / 合规）怎么定性和处理？

**合规 / 风控**
- 私募基金合规要点：**募集适当性 / 信息披露 / 关联交易 / 基金业协会（AMAC）登记备案**？
- **资管新规**对业务的影响？反洗钱 / KYC 流程？
- **三道防线**内控体系怎么搭？讲一个你处理过的风险 / 纠纷项目（STAR）。

**监管 / 资质**
- 持牌要求（法律职业资格 / 基金从业 / CFA）？
- 香港 SFC 牌照 **1 号（证券交易）/ 4 号（就证券提供意见）/ 9 号（资产管理）** 分别做什么？

## 六、风控 / 中后台

- 市场 / 信用 / 操作风险怎么识别计量？**VaR** 是什么、局限在哪？
- 投前 vs 投后风控分别做什么？一个风险事件的处置流程？

## 七、香港 / 外资岗（英文高频）

- *Walk me through a DCF. / Why this firm? / Tell me about a deal you followed.*
- 英文自我介绍 + 项目 STAR；中英术语切换（估值、尽调、合规的英文表达）。
