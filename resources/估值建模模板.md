# 估值建模模板 & 速查（DCF / 可比公司 / LBO）

> 金融求职、面试、实操常用三大估值法的**结构、公式、假设清单**——可直接照着搭 Excel。
> 📱 公众号 ［待填］ 回复 `估值` 领**含公式的 Excel 三件套（DCF / Comps / LBO）+ 更新**。欢迎 PR 把 Excel 补进 `resources/`。

## 一、DCF（现金流折现）

**步骤**
1. 预测期（5–10 年）自由现金流：`FCFF = EBIT×(1−税率) + 折旧摊销 − 资本开支 − 营运资本变动`
2. 折现率 **WACC** = `E/V×Re + D/V×Rd×(1−税率)`；其中 `Re = Rf + β×市场风险溢价`（CAPM）
3. 终值 **TV**：永续增长法 `FCFF_{n+1}/(WACC − g)` 或 退出倍数法 `EV/EBITDA × EBITDA_n`
4. 折现求和 → **企业价值 EV**；`股权价值 = EV − 净负债`；`每股价值 = 股权价值 / 股数`

**关键假设清单**：收入增速、EBIT 利润率、税率、资本开支/收入、营运资本/收入、WACC、永续增长 g、终值倍数。

**敏感性**：`WACC × g` 二维表；`WACC × 终值倍数` 二维表。

## 二、可比公司 / 可比交易（Comps）

- **选可比**：同行业、规模、成长性、盈利能力相近。
- **常用倍数**：EV/EBITDA、EV/EBIT、EV/Sales、P/E、P/B（金融股）、**P/AUM**（资管）。
- **步骤**：算每家可比倍数 → 取中位数/均值 → × 标的对应财务指标 → 得 EV / 股权价值。
- **可比交易**：看历史并购成交倍数（通常含**控制权溢价**，比可比公司法高）。

## 三、LBO（杠杆收购）

- **入场**：`入场 EV = EBITDA × 进入倍数`；资金来源 = 债务（约 3–6× EBITDA）+ 股权。
- **持有期**：5 年预测 EBITDA 与自由现金流，用 FCF **偿债去杠杆**。
- **退出**：`退出 EV = 退出 EBITDA × 退出倍数`；`退出股权 = 退出 EV − 剩余净债`。
- **回报**：`MOIC = 退出股权 / 入场股权`；`IRR`（年化）。
- **回报三来源**：去杠杆 / EBITDA 增长 / 倍数扩张。
- **关键假设**：进入与退出倍数、杠杆倍数、债务利率、EBITDA 增速、资本开支。

## 四、金融机构特殊估值（本库主打人群常用）

| 机构类型 | 主用方法 | 说明 |
|---|---|---|
| 银行 / 券商 | **P/B + ROE**、剩余收益模型 | 负债是"经营"不是"融资"，**不用 EV/DCF** |
| 公募 / 资管 | **P/AUM**、管理费收入倍数 | 盈利与 AUM 强相关 |
| 保险 | **内含价值 EV**、新业务价值 **NBV** | 长期负债特性，专用体系 |
| PE / VC（被投企业）| DCF + 可比 + 交易法交叉验证 | 早期看赛道/团队/可比融资 |
